# 기능 요구사항

## 1. 잔액 충전 API
- 기본기능
  - 금액을 충전하는 기능
- 정책
  - 충전금액은 0 이상이여야 한다.
<br/>
## 2. 잔액 조회 API
- 기본기능 
  - 보유 금액 조회 기능
<br/>
## 3. 주문 API
- 기본기능
  - 상품을 주문하는 기능
- 정책
  - 결제 실패시 상품 재고를 되돌리는 기능
  <br/>
## 4. 결제 API
- 기본기능
  - 상품을 결제하는 기능
- 정책
  - 할인쿠폰 적용시 할인적용 되는 기능
  - 상품 결제를 취소하는 기능
  <br/>
## 5. 선착순 쿠폰 API
- 기본기능
  - 상품에 대한 할인 쿠폰 제공 기능
- 정책
  - 선착순으로 쿠폰 제공 기능
  <br/>
## 6. 인기 판매 상품 조회 API
  - 최근 3일간 가장 많이 팔린 상위 5개 상품 정보를 제공
